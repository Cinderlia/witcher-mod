## 输入文件格式

每行一个字段，格式为 `字段名: 值`。以 `SCRIPT_FILENAME` 作为一条请求的起始标记，支持多条请求块。除 `SCRIPT_FILENAME` 外，其余字段均可省略。空值允许存在。

## 全局字段

- BASE_URL：当 SCRIPT_FILENAME 不是完整 URL 时，用该值补全为完整 URL
- loginSessionCookie：登录会话 Cookie 名称，可选
- INPUTSET：输入字典条目，可出现多行

## 请求块字段

- SCRIPT_FILENAME：必填，脚本路径或完整 URL
- GET：查询串，不含起始 `?`，留空表示无查询参数
- POST：请求体，留空表示无请求体
- COOKIE：Cookie 字符串，留空表示无 Cookie
- METHOD：请求方法，留空时根据 POST 是否为空自动选择 GET/POST
- HEADERS：请求头，使用 `;` 分隔多项，支持 `Key:Value` 或 `Key=Value`
- HEADER：单条请求头，支持 `Key:Value` 或 `Key=Value`，可重复多行
- RESOURCE_TYPE：资源类型，默认 `document`
- RESPONSE_STATUS：响应状态码，默认 0
- ATTEMPTS：请求尝试次数，默认 0
- PROCESSED：请求已处理次数，默认 0
- FROM：来源标记，默认 `ManualRequest`

## 示例

```
BASE_URL: http://localhost
loginSessionCookie: OpenEMR
INPUTSET: nv1=nv-val1

SCRIPT_FILENAME: /test.php
GET: a=1
POST: b=2
COOKIE: sess=abc
METHOD: POST
HEADERS: SOAPAction:http://x; HNAP_AUTH:xyz

SCRIPT_FILENAME: http://localhost/other.php
GET:
POST:
COOKIE:
```
